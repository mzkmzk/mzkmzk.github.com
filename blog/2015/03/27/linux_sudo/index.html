Linux 用户无法使用sudo

0x00问题

[oracle@localhost bin]$ sudo mkdir -p /opt/app/oracle
[sudo] password for oracle: 
oracle is not in the sudoers file.  This incident will be reported.

0x01解决办法

su root
echo '用户名  ALL=(ALL) ALL' >> /etc/sudoers
