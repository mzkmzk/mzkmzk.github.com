Git 入门

设置不要版本控制的文件

根目录下创建.gitignore

eg:

.gradle
/local.properties
/.idea/workspace.xml
/.idea/libraries
.DS_Store
/build

设置提交人和邮箱

仅作内容提醒所用

git config --global user.email "maizk@haplox.com"
git config --global user.name "Service"

设置密钥

常用命令

git init
git remote add origin ssh地址
git add .
git commit -m "first commit"
git branch 分支名称
git checkout 分支名称
git push origin 分支名称
