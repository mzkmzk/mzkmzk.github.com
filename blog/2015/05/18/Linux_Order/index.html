工具安装

ssh

  1. 安装客户端sudo apt-get install openssh-client
  2. 安装服务端sudo apt-get install openssh-server
  3. 开启服务/etc/init.d/ssh start
  4. 检测服务端口和开启情况ps -e |grep ssh
  5. 登陆sshssh username@ip
  6. 修改配置信息vim /etc/ssh/sshd_config
